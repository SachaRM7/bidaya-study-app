# بِدَايَةُ المُتَفَقِّه — Bidāyat al-Mutafaqqih

Application de révision interactive du Bidāyat al-Mutafaqqih de Wahīd b. ʿAbd al-Salām Bālī.

## Stack

- **Next.js 15** (App Router)
- **React 19**
- Google Fonts : Amiri, Source Serif 4, Outfit

## Contenu

4 chapitres de fiqh shāfi'ī :

- 💧 **Ṭahāra** — La purification (10 sections)
- 🕌 **Ṣalāt** — La prière (7 sections)
- 🌾 **Zakāt** — L'aumône obligatoire (1 section)
- 🌙 **Ṣiyām** — Le jeûne (1 section)

Chaque chapitre dispose de :

- **Fiches** — Les ḍawābiṭ en accordéon
- **Flashcards** — Révision question/réponse
- **Quiz** — Auto-évaluation

## Déploiement Vercel

```bash
npm install
npm run dev
```

Ou connectez le repo GitHub directement sur [vercel.com](https://vercel.com) — le déploiement est automatique.
